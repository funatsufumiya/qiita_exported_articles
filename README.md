# qiita_exported_articles

Qiitaの `@funatsufumiya` アカウント閉鎖に伴うバックアップです。

https://funatsufumiya.github.io/qiita_exported_articles/items/index.html によりウェブ閲覧が可能ですが、`img` にバックアップしてある画像一覧との紐づけができておらず、画像がリンク切れしている点に注意です。（現在少しずつ修正を試みています。）

## 補足（アカウント閉鎖の経緯）

一種のNo-AI活動の一環としてアカウントを閉鎖したものです。

私自身過去にCNN等のディープラーニングによる画像認識等を行っていたこともあり、AI利用自体は完全否定派ではありませんが、AI記事があまりに多く、業界全体としてもAI推進の動きが見られるため、その反発として閉鎖したものです。昔ながらの技術記事も多く存在することを理解しつつ、そして私自身過去にAI開発にコミットしているなかでの、苦渋の決断です。

ただ、AI自体の社会的意義や研究的意義は理解しているため、こちらのリポジトリに残っているデータのAI学習は禁止はしません。

[Cara](https://cara.app/) のような技術コミュニティがもしできれば、内容を吟味しながらそちらに記事を移すことは検討したいと思っています。
