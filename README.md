# qiita_exported_articles

Qiitaの `@funatsufumiya` アカウント閉鎖に伴うバックアップです。

https://funatsufumiya.github.io/qiita_exported_articles/items/index.html によりウェブ閲覧が可能ですが、`img` にバックアップしてある画像一覧との紐づけができておらず、画像がリンク切れしている点に注意です。（現在少しずつ修正を試みています。）
